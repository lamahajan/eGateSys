🔥 Step 1 — Create Firebase Project
1
Open Firebase Console
Go to console.firebase.google.com and sign in with your Google account
2
Create a New Project
Click "Add project" → Enter any name (e.g. LotusSchool) → Continue
3
Enable Realtime Database
Left menu → Build → Realtime Database → Create Database → Start in Test Mode → Done
4
Copy the Database URL
The format will be:
https://your-project-default-rtdb.firebaseio.com
5
Keep Rules Permanently Open (optional)
Paste this in Realtime Database → Rules tab so it does not expire after 30 days:
{ "rules": { ".read": true, ".write": true } }
Then click Publish ✅
📱 Step 2 — Connect the Apps
A
Student Management — Settings Tab
Open Student Management screen → Settings tab → enter password (mahajan) → enter Database URL and Sync Code → Save & Connect
B
Attendance — Firebase Setup
Attendance screen → Enter API Key and Database URL (copy from StaffFlow app) → Connect
C
Done! ✅
Use the same Firebase URL and Sync Code on both devices — data will sync automatically
