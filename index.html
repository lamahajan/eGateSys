📋 Bot Setup — Step by Step
Step 1 — Create a bot
1. Open @BotFather on Telegram
2. Send /newbot
3. Give it a name — e.g. Lotus Gate Pass
4. Give it a username — e.g. LotusGatePassBot (must end in "bot")
5. BotFather will send you a Token — paste it in the field above
Step 2 — Create a private channel
1. Create a new private Channel on Telegram
2. Add your bot as an Administrator with Post Messages permission
3. Send any message in the channel, then open this URL:
https://api.telegram.org/botYOUR_TOKEN/getUpdates 4. Copy the "id" value from the chat object (starts with -100...)

📋 IT Admin Setup — Step by Step
Step 1 — Tenant ID pata karo
1. School ka koi bhi Office 365 account se portal.azure.com kholo
2. Click your name in the top-right corner
3. Click Switch directory
4. Your Tenant ID (Directory ID) will be shown — copy it
Step 2 — ROPC Enable karo (IT Admin kare)
1. portal.azure.com pe jao → Azure Active Directory
2. App Registrations → All Applications
3. Search karo: Microsoft Office (App ID: d3590ed6...)
4. Authentication tab pe jao
5. Set "Allow public client flows" → Yes
6. Save ✅
Step 3 — MFA Disable karo (sirf is account ke liye)
1. Azure AD → Users → select the account that will be used for upload
2. Multi-Factor Authentication → set to Disabled
3. Or ask IT admin to create a dedicated service account with MFA disabled
⚠️ Use a dedicated account — do not use your main admin account. Create a separate account specifically for gate pass uploads.
